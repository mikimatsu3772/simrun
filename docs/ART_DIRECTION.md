# simrun — アートディレクション

## 決定事項(2026-06-13)

- **スタイル**: ゲームボーイ風(緑4色)
- **適用範囲**: ハイブリッド — キャラ・演出・ゲージ等は「GBスクリーン」パネル内にドット表現、ボタンやリストは現代的UIをGBパレットで着色
- **フォント**: DotGothic16(Google Fonts / OFL)を全面適用

## GBパレット(この4色+透明以外は原則使わない)

| 名前 | HEX | 用途 |
|---|---|---|
| `gbDarkest` | `#0F380F` | 線・文字・スプライト輪郭、アプリ背景 |
| `gbDark` | `#306230` | スプライト中間色、カード背景 |
| `gbLight` | `#8BAC0F` | アクセント、ボタン、ゲージ |
| `gbLightest` | `#9BBC0F` | スクリーンパネル背景、明色文字 |

## UI構成ルール

- アプリ全体の背景は `gbDarkest`、カードは `gbDark`、本文文字は `gbLightest`
- ランナーなどのドット絵は `gbLightest` の「スクリーンパネル」(角丸+内側に少し余白)に載せ、
  スプライトは `gbDarkest` / `gbDark` / `gbLight` の3色で描く(ゲームボーイ実機の見え方)
- ステータス等の色分けは行わない(4色縛り)。差は長さ・形・ラベルで表現する
- 例外: エラー表示等、OS標準が期待される箇所のみマテリアル既定色を許容

## スプライト規約

- ランナーは 16×16。文字マップ(`.`=透明, `D`=gbDarkest, `M`=gbDark, `L`=gbLight)で
  `lib/ui/pixel/` にコードとして定義し、CustomPainter で描画(画像アセット不要、動的に拡大可能)
- 走りアニメは2フレーム、240ms間隔
- 拡大は整数倍のみ(にじみ防止)、`FilterQuality.none`
